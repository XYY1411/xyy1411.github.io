# XYY1411 个人主页

基于 [Aerial by HTML5 UP](https://html5up.net/aerial) 模板的 GitHub Pages 单页面个人主页。

## 网址

🌐 [www.xyy1411.top](https://www.xyy1411.top/)

## 技术栈

- 纯静态 HTML + CSS + JS
- [Font Awesome 5.9](https://fontawesome.com/) 图标
- [Workbox v6](https://developer.chrome.com/docs/workbox/) Service Worker 离线缓存
- GitHub Pages + Cloudflare CDN 部署

## 本地开发

```bash
# 启动本地服务器
python -m http.server 8765
# 浏览器访问 http://127.0.0.1:8765
```
